# footballer
footballer app

* Check the players from the major leagues
* Save your favourite teams

## Live

[Live](http://footballer-eu.azurewebsites.net)

## Run

```bash
npm install
```

## Tests

```bash
gulp test
```
or

```bash
gulp test:auto
```

## Developer mode

gulp serve

```bash
gulp serve
```
or

```bash
gulp serve:dist
```


## Distribution

```bash
gulp
```
